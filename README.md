# rolebaselogin
A role base login system

## Get started
# Client
**`cd client`**

**`npm i`**

**`npm start`**


# Server
**`cd server`**

**`npm i`**

**`npm start`**

### Description
*Role base authentication application can authenticate user (i.e. user, admin, etc) and respond according to it. User can register as default user or admin*
